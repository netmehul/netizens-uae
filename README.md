# Scale & Launch System - UAE Real Estate Landing Page

This is a high-performance marketing landing page and modular launch system built specifically for the Dubai real estate market. 

It is designed to be highly scalable, allowing rapid deployments ("Build Once, Scale Across Projects") for different properties or agencies without the need to start from scratch each time.

## Key Features

- **Performance First**: Built with modern, lightweight technologies for optimal SEO and conversion speeds.
- **Modular Components**: Reusable sections (Hero, Case Studies, Problem / Solution, Comparison Tables) make spinning up new pages take 3-5 days instead of weeks.
- **Dubai Focused**: Tailored specifically for the high-intent lead generation strategies required in the competitive UAE real estate market.
- **Premium Aesthetics**: Sophisticated UI/UX matching the exact expectations of luxury real estate buyers.

## Tech Stack

This project is built using:

- **Framework:** React.js + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (with custom tokens and utilities for a unique brand identity)
- **Animations:** Custom CSS and GSAP (GreenSock Animation Platform) 
- **Icons:** Lucide React & Custom SVGs

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository** (or download the source code):
   ```bash
   git clone https://github.com/netmehul/AG-project.git
   cd AG-project
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   This will start the Vite dev server, usually available at `http://localhost:5173`. Make edits and see them updated in real-time.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The output will be generated in the `dist/` directory, ready to be deployed to any static hosting service (Vercel, Netlify, AWS S3, etc.).
